# 莫奈的画境 / Monet's Dreamscape

印象派风格横版卷轴探索游戏 — 纯 HTML5 Canvas 实现。探索莫奈的六幅名画，收集隐藏的宝物，见证莫奈与卡米耶的爱情故事。

## 简介

玩家操控一个发光的精灵，在以莫奈真实画作为蓝本的世界中冒险。每个关卡都有独特的游戏机制，灵感来自画作主题 — 从灯光摇曳的巴黎画室，到风吹花舞的花园，再到撑着阳伞滑翔过山丘。

游戏按时间线讲述莫奈与卡米耶·东西欧的爱情故事，配有英文语音旁白和中英双语字幕，从他们在巴黎画室的初遇，延续到莫奈晚年在吉维尼的岁月。

## 六大关卡

| # | 画作 | 年份 | 独特机制 |
|---|------|------|----------|
| 1 | 穿绿裙的女人 | 1866 | **光影探索** — 玩家携带灯笼，黑暗区域隐藏平台，靠近才能照亮 |
| 2 | 花园中的女人们 | 1866 | **风力系统** — 阵风周期性吹拂，推动玩家并摇晃平台 |
| 3 | 撑阳伞的女人 | 1875 | **阳伞滑翔** — 长按跳跃键滑翔，上升气流区域可向上弹射 |
| 4 | 睡莲 | 1906 | **消失的莲叶** — 踩上平台后会逐渐消失 |
| 5 | 日出·印象 | 1872 | **潮汐循环** — 水位周期性升降，淹没低处平台 |
| 6 | 日本桥 | 1899 | **季节变换** — 环境从春天过渡到夏天再到秋天 |

## 功能特性

- **6个关卡** — 基于莫奈真实画作，每关独特游戏机制
- **中英双语** — 菜单和游戏内均可实时切换
- **语音旁白** — 莫奈与卡米耶的对话（42条语音，英文TTS + 中英字幕）
- **画作渐现** — 背景从灰色素描逐渐变为全彩，随收集进度展现
- **程序化音乐** — Web Audio API 实时生成环境音乐，无需音频文件
- **收集系统** — 每关3个颜料管钥匙 + 5个纪念品，持久化图鉴
- **画作画廊** — 查看已解锁的莫奈名画及介绍
- **AI生成精灵** — 8帧角色动画，运行时自动去除黑色背景
- **3层视差滚动** — 雾气、光柱、暗角、噪点等氛围特效
- **移动端适配** — 响应式布局 + 虚拟触控按钮

## 架构亮点

### 单文件游戏引擎
所有游戏逻辑集中在 `game.js` 一个文件中 — 物理引擎、渲染、关卡数据、国际化、音频、UI 状态管理。这是为了实现零依赖的静态托管部署。

### 精灵处理管线
AI 生成的角色贴图带有黑色背景，通过运行时两遍像素处理自动去除：
1. **第一遍**：扫描所有帧，找到内容边界框，计算最大统一尺寸
2. **第二遍**：将每帧居中放入统一画布，移除暗色像素并做渐变透明过渡

确保所有动画帧尺寸一致，防止状态切换时的抖动。

### 画作渐现系统
三种揭示模式增加关卡间的视觉多样性：
- **全局渐变** — 整个背景从去饱和逐渐变为全彩
- **局部扩散** — 颜色从每个收集点向外辐射
- **分层揭示** — 水平区域从下到上逐层显现

### 程序化音乐
不包含任何音乐文件，通过 Web Audio API 实时生成环境音乐：
- 第1层：和弦铺底（持续的和声基础）
- 第2层：五声音阶旋律（随机生成）
- 第3层：低音（和弦根音）
- 第4层：滤波白噪声纹理

## 设计理念

视觉设计取材于美术馆美学 — 暖色羊皮纸色调（#f5ead0）用于文字，深色画廊黑（#0a0a12）用于背景。Georgia 衬线字体呼应古典艺术图录的气质。大量运用透明度叠加和模糊滤镜，再现印象派绘画特有的光影朦胧感。

叙事沿莫奈与卡米耶的真实时间线展开，将平台跳跃机制植根于情感背景之中，而非纯粹的难度堆砌。
